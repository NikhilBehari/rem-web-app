# rem web app
Website to accept donations for the rem app. 

## Get Started 
This project is built with React. To install dependencies, run: 
```javascript
yarn install
```
in the root directory. To run a locally, run: 
```javascript
yarn start
```

## Project Infrastructure 

```
index.js: Root 
  |
  |--App.js: App HomePage
      |
      |--Pages/Donate.js: Donation panel
```
